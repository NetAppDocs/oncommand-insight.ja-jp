= アセットへのアプリケーションの割り当て
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
アプリケーションを定義したら、それらのアプリケーションを特定のアセットに関連付ける必要があります。簡単なアドホック方式を使用して、アプリケーションをアセットに適用できます。アプリケーションを一括で適用する場合は、クエリメソッドを使用して、アプリケーションに割り当てるアセットを特定する必要があります。
