= ストレージプールを確認します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ストレージプールのランディングページを確認すると、関連するアセットと同じIOPSスパイクが表示されます。[Resources]セクションでは、このストレージプールのランディングページが旅行アプリケーションのボリュームにリンクしていることを確認できます。ボリュームをクリックすると、ボリュームのランディングページが開きます。
