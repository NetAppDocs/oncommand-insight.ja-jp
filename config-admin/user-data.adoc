= ユーザデータ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
アセットページの[User Data]セクションには、ユーザが定義したアプリケーション、ビジネスエンティティ、アノテーションなどのデータが表示されます。

仮想マシンのアセットページの[User Data]セクションにアプリケーション、ビジネスエンティティ、およびアノテーションが割り当てられている場合の表示例を次に示します。
