= アノテーションタイプの管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
OnCommand Insight には、アセットのライフサイクル（開始日や終了日）、建物やデータセンターの場所、階層など、カスタマイズしてレポートに表示できるデフォルトのアノテーションタイプがいくつか用意されています。デフォルトのアノテーションタイプの値を定義することも、独自のカスタムアノテーションタイプを作成することもできます。これらの値は後で編集できます。
