= アセットを照会しています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クエリを使用すると、環境内のアセットをユーザが選択した条件（アノテーションとパフォーマンス指標）に基づいてきめ細かく検索することで、ネットワークの監視とトラブルシューティングを行うことができます。また、アセットにアノテーションを自動的に割り当てるアノテーションルールにはクエリが必要です。
